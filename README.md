# Perpetuo Flow

Demo estética y funcional de un POS conversacional para cafeterías.

## Qué hace
- Chat visible para el cliente
- Interpretación simple de pedidos en español
- Resumen de orden
- Ticket para barista
- Flujo de pago demo en $MON

## Run local
Abre `index.html` en el navegador.

## Nota
El flujo blockchain es una simulación visual para la demo del hackathon.
